Medical Data Visualizer Project

This project uses Pandas, Seaborn, and Matplotlib to analyze and visualize medical examination data.

Files:
- medical_data_visualizer.py : main code
- main.py : runs the visualizations
- test_module.py : unit tests
- medical_examination.csv : dataset

Run with:
    python3 main.py

Outputs:
- catplot.png
- heatmap.png
